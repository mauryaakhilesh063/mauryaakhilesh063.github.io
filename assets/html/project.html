Project

🏦 CLOS Integration Project – Newgen

Overview :
I contributed to the integration of Newgen’s Commercial Lending Origination System (CLOS), a platform designed to streamline and digitize the commercial loan lifecycle—from application to disbursement.

My Role

  • Developed and optimized JavaScript modules for dynamic form rendering and validation.
  • Implemented matrix-based workflows to handle multi-stage loan approvals.
  • Debugged and enhanced performance of document upload and PDF embedding features.

Key Highlights

  • Improved turnaround time by 30% through automation of manual underwriting steps.
  • Integrated REST APIs for real-time credit checks and document verification.
  • Ensured scalability by modularizing code and optimizing DOM interactions.

Tech Stack

JavaScript, HTML/CSS, REST APIs, PDF.js, GitHub, Agile (Scrum)

Impact

The project significantly reduced loan processing time and improved user experience for both internal teams and external applicants. It also laid the groundwork for future integrations with banking CRMs and compliance systems.
